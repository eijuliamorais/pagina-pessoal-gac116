# Página Pessoal - Programação Web

Repositório destinado à **Atividade Prática 2** da disciplina de Programação Web (GAC116).

O objetivo do projeto é desenvolver uma página pessoal utilizando HTML, CSS e JavaScript, implementando layouts responsivos e alternância de temas (Claro/Escuro) através de dois frameworks CSS distintos.

## 🌐 Visualizar o Projeto (GitHub Pages)

Acesse a página publicada clicando no link abaixo:

**[Acessar minha Página Pessoal](https://eijuliamorais.github.io/pagina-pessoal-gac116/)**

> **Nota:** Para ver a versão construída com o framework Bulma, basta adicionar `/versao2.html` ao final da URL acima.

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Estruturação semântica)
* **CSS3** (Estilização personalizada e transições)
* **JavaScript** (Lógica do botão toggle de tema Claro/Escuro)
* **Bootstrap 5** (Framework CSS - Versão Principal)
* **Bulma** (Framework CSS - Segunda Versão)
